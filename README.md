# 📚 書籍Cコードリーダー

書籍のCコードバーコードを読み取り、書籍の分類情報を表示するシンプルなWebアプリケーションです。

## 🎯 機能

- Cコードバーコードの読み取り（バーコードリーダー対応）
- Cコードの解析と分類情報表示
  - 販売対象（一般、教養、専門など）
  - 発行形態（単行本、文庫、新書など）
  - 内容分類（文学、歴史、科学など）

## 🚀 使い方

### 1. アプリケーションの起動

`index.html` をブラウザで開くだけで使用できます。

```bash
# ファイルをダブルクリック、または
start index.html
```

### 2. Cコードの読み取り

1. バーコードリーダーを接続
2. 入力欄にフォーカスがある状態で書籍のCコードバーコードをスキャン
3. 自動的に分類情報が表示されます

### 3. 手動入力

バーコードリーダーがない場合は、4桁のCコードを手動で入力できます。

**テスト用サンプルCコード:**
- `0091` - 一般・文庫・日本文学
- `0293` - 一般・新書・日本歴史
- `3310` - 専門・単行本・総記
- `4744` - 一般・絵本・絵本

## 📋 Cコードについて

Cコードは日本の書籍に付けられる4桁の分類コードです：

- **1桁目**: 販売対象（0=一般、1=教養、2=実用、3=専門、8=児童など）
- **2桁目**: 発行形態（0=単行本、1=文庫、2=新書、3=全集など）
- **3-4桁目**: 内容分類（00=総記、10=哲学、20=歴史、90=文学など）

### Cコードの位置

日本の書籍には通常、以下のバーコードが印刷されています：
1. ISBNバーコード（13桁）
2. **Cコードバーコード（4桁）** ← このアプリで対象

Cコードバーコードは通常、書籍の裏表紙や奥付にISBNとは別に印刷されています。

## ⚠️ 注意事項

- Cコードは4桁の数字である必要があります
- 対応しているのは日本の書籍のCコードです
- 実際の書籍で動作確認してください

## 🔧 カスタマイズ

### Cコード分類の追加・修正

`ccode-data.js` ファイルの `CCODE_DATA` オブジェクトを編集することで、分類を追加・修正できます。

```javascript
const CCODE_DATA = {
    target: { ... },
    format: { ... },
    content: { ... }
};
```

### デザインの変更

`style.css` を編集してデザインをカスタマイズできます。
